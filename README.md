# 🎓 SVPCET College Website

A multi-page college website built for the **AI Department of Sri Venkatesa Perumal College of Engineering and Technology (SVPCET), Puttur**.

---

## 📌 About the Project

This project is a front-end college website developed using pure HTML and CSS. It was built as part of my academic web development practice to demonstrate skills in structuring multi-page websites with navigation, styling, and responsive layout.

---

## 🗂️ Pages Included

| Page | File | Description |
|------|------|-------------|
| Home | `index.html` | Landing page with header, nav, and hero section |
| About Us | `AboutUsPage.html` | College overview and department info |
| Courses | `CoursesPage.html` | Courses offered by the department |
| Student Info | `StudentInfo.html` | Student-related information |
| Contact | `ContactPage.html` | Contact and syllabus details |
| Login | `loginpage.html` | Student login page |
| Register | `registerpage.html` | Student registration page |

---

## 🛠️ Tech Stack

- **HTML5** — Page structure and content
- **CSS3** — Styling, transitions, hover effects, flexbox layout

---

## ✨ Features

- Responsive header with navigation links
- Hero section with gradient overlay
- Hover animations on nav links (scale + color transition)
- Multi-page navigation linking all pages together
- Login and Register pages for student access
- Clean color theme using SVPCET branding (navy blue `#003366`)
