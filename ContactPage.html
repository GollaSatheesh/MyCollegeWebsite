Artificial Intelligence Department

Final Year B.Tech - Syllabus

Subjects

1. Distributed Database (DDB)

  • Introduction to Distributed Databases
  • Data Fragmentation and Allocation
  • Query Processing and Optimization
  • Concurrency and Recovery

2. Natural Language Processing (NLP)

  • Text Preprocessing and Tokenization
  • POS Tagging and Named Entity Recognition
  • Syntax and Semantics
  • Text Classification and Sentiment Analysis

3. Cryptography and Network Security (CNS)

  • Symmetric and Asymmetric Cryptography
  • Block Ciphers and RSA
  • Message Authentication and Hash Functions
  • Firewalls and Intrusion Detection

4. Digital Forensics (DF)

  • Introduction to Forensics
  • Data Acquisition and Analysis
  • File Systems and Memory Forensics
  • Incident Response and Investigation

5. Power BI

  • Data Loading and Cleaning
  • Data Modeling and DAX
  • Interactive Dashboards
  • Publishing and Sharing Reports

6. SAP

  • SAP Overview and Architecture
  • ERP Modules and Navigation
  • Business Processes in SAP
  • Reporting and Analytics

7. ABAP (Advanced Business Application Programming)

  • ABAP Syntax and Data Types
  • Internal Tables and Loops
  • Modularization Techniques
  • Reports and Forms
